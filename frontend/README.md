# 校园流浪猫管理系统 🐱

基于 Vue 3 的校园流浪猫智能百科与社区管理工具

## 📋 项目概述

这是一个帮助校园流浪猫管理的微信小程序，主要功能包括：
- AI智能问答
- 拍照识猫
- 投喂记录管理
- 猫咪档案管理

## 🛠 技术栈

- **前端框架**: Vue 3
- **构建工具**: Vite
- **路由**: Vue Router 4
- **状态管理**: Pinia
- **UI**: 自定义移动端样式

## 📁 项目结构

```
src/
├── assets/          # 静态资源
├── components/      # 通用组件
├── pages/          # 页面组件
│   ├── Home.vue    # 首页
│   ├── AiChat.vue  # AI问答
│   ├── Feeding.vue # 投喂统计
│   ├── Profile.vue # 个人中心
│   ├── FeedingAdd.vue # 新建投喂
│   └── CatDetail.vue # 猫咪详情
├── router/         # 路由配置
├── store/          # 状态管理
└── App.vue         # 根组件
```

## 🚀 开发指南

### 环境准备

1. **安装依赖**
   ```bash
   npm install vite @vitejs/plugin-vue
   ```

2. **启动开发服务器**
   ```bash
   npm run dev
   ```

3. **构建生产版本**
   ```bash
   npm run build
   ```

### 页面说明

#### 首页 (Home.vue)
- 猫咪表情包展示
- 投喂建议卡片
- AI问答入口
- 底部导航栏

#### AI问答 (AiChat.vue)
- 对话式问答界面
- 消息历史记录
- 输入框和发送功能

#### 投喂记录 (Feeding.vue)
- 个人投喂统计数据
- 投喂历史列表
- 新建投喂入口

#### 个人中心 (Profile.vue)
- 用户信息和头像
- 成就徽章系统
- 个人数据统计
- 功能菜单

#### 新建投喂 (FeedingAdd.vue)
- 猫咪选择下拉框
- 食物类型选择
- 投喂量输入
- 过度投喂预警

#### 猫咪详情 (CatDetail.vue)
- 猫咪基本信息展示
- 详细档案信息
- 投喂历史时间轴
- 投喂和分享功能
