- **后端服务**: 🟢 运行中 (http://localhost:8000)

## 🎯 项目特色

- **一套代码，多端发布**：支持微信小程序 + App
- **Vue 3 语法**：现代化的开发体验
- **组件化架构**：易于维护和扩展
- **移动端优先**：专为移动设备优化

## 📁 项目结构

```
uniapp/
├── pages/           # 页面组件
├── components/      # 通用组件
├── store/          # 状态管理
├── utils/          # 工具函数
├── static/         # 静态资源
├── manifest.json   # 应用配置
└── pages.json      # 页面配置
```

## 🚀 开发指南

### 环境准备
1. 安装 HBuilderX 或 VS Code + uni-app 插件
2. 导入项目到开发工具
3. 运行项目

### 页面说明
- 🏠 首页 - 猫咪发现和投喂建议
- 💬 AI问答 - 智能对话功能
- 📊 投喂统计 - 数据展示
- 👤 个人中心 - 用户信息
- ➕ 新建投喂 - 记录表单
- 🐱 猫咪详情 - 档案信息

## 🎨 设计规范

- 移动端优先设计
- 响应式布局
- 优雅的用户体验
- 科学的数据管理

## 🔧 技术栈

- **框架**: uni-app + Vue 3

- **UI组件**: 自定义组件
- **构建工具**: HBuilderX
